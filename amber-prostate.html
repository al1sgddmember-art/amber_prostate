はじめまして — ここはサンプルページです

このファイルをそのまま `index.html` として置けば GitHub Pages や Netlify にポン置きで公開できます。

シンプル

余計な依存無し。HTML/CSS/JS をそのまま置くだけでOK。

レスポンシブ

スマホでも読めるレイアウト。必要なら Tailwind 等に差し替えても良い。

カスタマイズ自在

ロゴ、色、文言を編集して自分用に仕上げてください。

使い方メモ

  1. このファイルを `index.html` として保存
  2. GitHub リポジトリにアップロード（public）
  3. Settings → Pages で `main` / root を選択して保存
  4. 表示されたURLにアクセスして確認
